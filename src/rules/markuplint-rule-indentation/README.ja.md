# インデント (`indentation`)

インデントが統一されていないと警告します。

## ルールの詳細

👎 間違ったコード例

```html
<div>
  <span>lorem</span>
	ipsam
   </div>
```

👍 正しいコード例

```html
<div>
	<span>lorem</span>
	ipsam
</div>
```

### オプション

設定値|デフォルト|解説
---|---|---
`"tab"`|✓|タブ文字に統一します。
[数値]||指定した数値の幅のスペース文字（正の整数）。

### デフォルトの警告レベル

`warning`
